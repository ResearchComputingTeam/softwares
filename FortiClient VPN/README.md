# FortiClient VPN

## Overview

FortiClient VPN is a secure VPN client required for users to establish an authenticated remote connection to the HPC systems managed by RCCG. This directory is maintained for reference purposes and contains bundled installers for supported platforms.

## Bundled Installers

Pre-packaged installers are available in the following subdirectories:

| Platform | Directory | File(s) |
|---|---|---|
| macOS | [`MacOs/`](./MacOs/) | `FortiClientVPN_7.4.3.6667_OnlineInstaller.dmg` |
| Windows (x86-64) | [`Win/`](./Win/) | `FortiClientVPNInstaller.exe` — for standard Intel/AMD 64-bit processors (most desktop and laptop PCs) |
| Windows (ARM64) | [`Win/`](./Win/) | `FortiClientVPNInstaller_arm64.exe` — for ARM-based Windows devices (e.g., Microsoft Surface Pro X, Snapdragon-powered laptops) |

> **Note:** The installers stored here may not reflect the latest release. It is recommended to download the most recent version directly from the official Fortinet website (see below).

## Downloading the Latest Version

To obtain the latest **FortiClient VPN-only** installer appropriate for your operating system:

1. Visit the official Fortinet Product Downloads page:
   **https://www.fortinet.com/support/product-downloads**
2. Locate **FortiClient VPN** in the product list.
3. Select your operating system and architecture.
4. Download the **FortiClient VPN Only** package (not the full FortiClient suite).

## Installation

Follow the standard installation procedure for your operating system after downloading the installer. No license key is required for the VPN-only client.
